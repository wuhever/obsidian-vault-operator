# 09_Reference — READ-ONLY

This folder is a **read-only archive**.

## Rules

- **AI agents MAY NOT** create, modify, or delete files in this folder
- **Only the owner** (manually) adds reference material here
- If you want to build on a reference, create a new note in the relevant working folder (client's `Research/`, `03_Projects/`, `07_Ideas/`, etc.) and link back with `[[wikilinks]]`

## What Goes Here

- Saved articles, papers, external documents
- Canonical references you want to preserve unchanged
- Archived projects, completed work for historical reference

## What Does NOT Go Here

- Active working notes
- Anything agents should be able to modify
- Duplicates of content that lives elsewhere
