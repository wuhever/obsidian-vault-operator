## Tokens

| Token | What it means | Example value | Where it appears |
|---|---|---|---|
| `{{VAULT_DIR}}` | Name of the vault directory on disk (default: `vault`) | `phd-vault` | Replaces the `vault/` directory name if renamed |
| `{{VAULT_NAME}}` | Short slug/name for the vault | `sarah-phd-vault` | `{{VAULT_DIR}}/README.md`, `AGENTS.md`, `operator-prompt.md` |
| `{{OWNER_NAME}}` | Display name of the vault owner | `Sarah` | `operator-prompt.md` (Key People), `{owner-slug}-space/_README.md` |
| `{{OWNER_SLUG}}` | Lowercase/hyphenated owner name | `sarah` | Folder name `{owner-slug}-space/` |
| `{{VAULT_PURPOSE}}` | One-sentence description | `PhD research on marine biology` | `AGENTS.md`, `README.md` |
| `{{VAULT_TYPE}}` | Type picked during interview | `researcher` | `AGENTS.md`, `README.md`, `operator-prompt.md` |
| `{{ORG_OR_TOPIC}}` | Short phrase for "Relevance to ___" | `your research` | `_Templates/research-note.md` |
| `{{VAULT_PATH}}` | Absolute path to the vault directory | `C:\Users\sarah\Documents\sarah-phd-vault\phd-vault` | `operator-prompt.md`, `vault-operator/CLAUDE.md` |
| `{{VAULT_OPERATOR_PATH}}` | Absolute path to `vault-operator/` | `C:\Users\sarah\Documents\sarah-phd-vault\vault-operator` | Same |
| `{{SCHEME_TABLE}}` | Generated markdown table of created folders + purposes | *(a multi-line table)* | `{{VAULT_DIR}}/AGENTS.md`, `{{VAULT_DIR}}/README.md`, `vault-operator/operator-prompt.md` |

## Derivation rules

- `{{VAULT_DIR}}`: user-provided, default `vault`. Must be filesystem-safe (no spaces).
- `{{OWNER_SLUG}}`: lowercase `{{OWNER_NAME}}`, spaces â†’ hyphens, strip special chars. "Dr. Ali Al-Hameiri" â†’ `ali-al-hameiri`.
- `{{ORG_OR_TOPIC}}`: inferred from `{{VAULT_PURPOSE}}` + `{{VAULT_TYPE}}`:
  - researcher â†’ `your research`
  - student â†’ `your studies`
  - business â†’ `your work`
  - hobbyist-builder â†’ `your projects`
  - personal â†’ `your interests`
  - custom â†’ ask user
- `{{VAULT_PATH}}` = repo_root + "/" + `{{VAULT_DIR}}`
- `{{VAULT_OPERATOR_PATH}}` = repo_root + "/vault-operator"
- `{{SCHEME_TABLE}}`: generated by the wizard in Phase 4d, formatted as a markdown table.
